# Control Flow Assignment

This folder contains the solutions for the Python Control Flow assignment.

## Files Included
* `main.py`: The main executable Python script.
* `README.md`: This documentation file.

## How to Run
1. Ensure Python 3 is installed on your system.
2. Open your terminal or command prompt.
3. Navigate to the directory containing these files.
4. Execute the script by running: `python control_flow_assignment.py`

## Task Execution Order
For convenience, all tasks are combined into a single script. When you run the file, the tasks will execute in the following order:

1. **Task 1 (Discount Rules):** The console will prompt you to input a single integer order amount to demonstrate the `if/elif/else` logic.
2. **Task 2 (Multiple Orders):** The script will automatically process a hardcoded list of orders using a `for` loop and print a summary table.
3. **Task 5 (Loop Control):** The script will automatically process a list of daily sales to demonstrate the `break` and `continue` keywords.
4. **Task 3 (User Menu):** Finally, an interactive `while` loop menu will appear. You can add new orders, view their totals with applied discounts, or press `q` to terminate the script entirely.
