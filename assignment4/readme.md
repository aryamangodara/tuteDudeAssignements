# Python File Handling Assignment

## Project Overview
This project demonstrates foundational file handling operations in Python. It is divided into 7 distinct tasks that cover writing to files, appending data, parsing file contents, aggregating data, safely reading files using OS safeguards, and writing formatted reports based on user input. 

### Features Covered:
* Creating and writing to `.txt` files.
* Understanding the difference between `.read()`, `.readline()`, and `.readlines()`.
* Appending data using the `"a"` file mode.
* Data parsing and type casting (converting string lines to integers).
* Safe file reading using `os.path.exists()` to prevent crash errors.
* Accepting user input to dynamically generate text files and dictionary-based discount reports.

## Prerequisites
* Python 3.x installed on your machine.

## How to Run the Project
1. Download or clone the repository to your local machine.
2. Ensure you have the main script named `file_handling_assignment.py` in your working directory.
3. Open a terminal or command prompt.
4. Navigate to the folder containing the script using the `cd` command.
5. Run the script by executing the following command:
   ```bash
   python file_handling_assignment.py
