# Python Functions Assignment

This project contains a single Python script (`functions_assignment.py`) that demonstrates proficiency with basic functions, recursion, lambda expressions, `map()`, `filter()`, and basic command-line menu logic in Python.

## Prerequisites
To run this script, you must have **Python 3.x** installed on your system. 

## Project Files
* `functions_assignment.py`: The main script containing solutions for Tasks 1 through 7.
* `README.md`: This file, containing instructions on how to run the project.

## How to Run

1. Open your terminal or command prompt.
2. Navigate to the directory where you saved the files.
   ```bash
   cd path/to/your/folder
   ```
3. Run the script using Python:

Bash
```python functions_assignment.py```

(Note: Depending on your system configuration, you might need to use python3 instead of python)
